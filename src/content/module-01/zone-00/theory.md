## Tu primer hechizo: `print()`

`print()` le dice a Python qué mostrar en pantalla. Es la forma más simple
de ver el resultado de tu código.

```python
print("Hola, Pythonia!")
```

Ejecutá ese código y verás el mensaje aparecer. Eso es un **programa**.

## Los comentarios `#`

Las líneas que empiezan con `#` son comentarios. Python las ignora —
son notas para vos (o para el próximo mago que lea tu código).

```python
# Esto es un comentario, Python no lo ejecuta
print("Esto sí se ejecuta")
```

## Varias líneas, varias instrucciones

Podés escribir varios `print()` uno debajo del otro. Se ejecutan en orden, de arriba hacia abajo.

```python
print("Línea 1")
print("Línea 2")
print("Línea 3")
```

---

> **Consejo del grimorio:** `print()` acepta texto entre comillas simples `'...'`
> o dobles `"..."`. Ambas funcionan igual.
